# Corporate Valentine Week Website

A humorous, corporate-themed reinterpretation of Valentine's Week for the modern workplace.

**Live Website:** [https://aman2241.github.io/corporate-valentine/](https://aman2241.github.io/corporate-valentine/)

## Overview

This project reimagines the traditional days of Valentine's Week through the lens of corporate culture, HR policies, and developer humor. Built with Vanilla HTML, CSS, and JavaScript.
